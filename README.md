# 📘 e2idic

e2idic is a simple **web-based Ilokano ↔ English dictionary** that allows users to search, browse, and explore translations easily. It uses local data for fast performance and supports **Progressive Web App (PWA)** features.

---

## 🚀 Features

- 🔍 Real-time search (shows only matched results)  
- 📚 Dictionary loaded from local JSON file  
- 📄 Pagination for easy browsing  
- 📱 Responsive design (mobile-friendly)  
- ⚡ Works offline using Service Worker  
- 📲 Installable as a PWA  

---

## 🛠️ Built With

- HTML  
- CSS

- JSON (dictionary data)

## ▶️ Getting Started

### 1. Clone the repository
git clone https://github.com/ermon07/e2idic.git


### 2. Run the project

Open `index.html` in your browser.

> ⚠️ For full PWA features (offline mode), use a local server (e.g., VS Code Live Server).

---

## 💡 How It Works

- Loads data from `ilocano_dictionary.json`  
- Displays words using `script.js`  
- Filters results through search  
- Uses pagination for navigation  
- Caches files using `service-worker.js`  
- Enables install via `manifest.json`
