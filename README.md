```
cd egg-timer
```

## 🌐 Usage

1. Select an egg type from the dropdown
2. Click **Start** to begin the timer
3. Use:
   - **Pause** to stop temporarily
   - **Reset** to clear the timer
4. (Optional) Enter custom minutes and click **Set**

## 💡 How It Works

- Loads data from `ilocano_dictionary.json`  
- Displays words using `script.js`  
- Filters results through search  
- Uses pagination for navigation  
- Caches files using `service-worker.js`
