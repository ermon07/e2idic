# 🍳 Egg Timer Web App

A simple and responsive **Egg Timer Web Application** built using **HTML, CSS, JavaScript, and Bootstrap 5**. This app helps you perfectly time your eggs with preset options or custom durations.

---

## 🚀 Features

- 🍳 Preset egg timers:
  - Soft Boiled (5 minutes)
  - Medium Boiled (7 minutes)
  - Hard Boiled (10 minutes)
- ⏱ Real-time countdown timer (MM:SS format)
- ▶️ Start, Pause, and Reset controls
- 🛠 Custom timer input (user-defined minutes)
- 🎨 Responsive UI using Bootstrap 5

## 🛠 Installation & Setup

1. Clone the repository:
```
git clone https://github.com/your-username/egg-timer.git
```

2. Navigate to the project folder:
```
cd egg-timer
```

3. Open `index.html` in your browser

---

## 🌐 Usage

1. Select an egg type from the dropdown
2. Click **Start** to begin the timer
3. Use:
   - **Pause** to stop temporarily
   - **Reset** to clear the timer
4. (Optional) Enter custom minutes and click **Set**
